Featured Module: three.js Material and Lighting Demo

Here we use three.js to create a glass icosahedron with a slight frosting effect. The wireframe is added by rendering a thin cylinder across each vertex. (WebGL's built-in wireframes cannot consistently render lines of variable width, modeling them as cylinders looks much better). We then add three colored lights to the plane perpendicular to the camera, angled at 120 degrees. Using the Physical Material feature, we add a frosted effect, a thickness, and a high index of refractivity to accentuate the refraction of the wireframe. The colored light trick is much more obvious if we leave specular highlights alone, but if we set them to 0, we won't get any pure tone glare. The final code can be inspected here. I'm excited to use these techniques for data visualization and other toys in the future.
